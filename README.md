# ShopHub E-Commerce Platform

This repository contains design documentation and code examples for the ShopHub e-commerce platform.

## Repository Structure

### Task_1 - Class Diagram
Contains the class diagram documentation explaining the structure of the e-commerce system, including six main classes: User, Cart, Product, Category, Order, and Payment.

**Files:**
- `class-diagram.jpg` - Visual class diagram
- `class-diagram.md` - Detailed explanation of classes, attributes, methods, and relationships

### Task_2 - UML Diagrams
Contains documentation for three types of UML diagrams that show how the system works.

**Files:**
- `use-case-diagram.md` - Shows what users (Customer, Admin) can do in the system
- `sequence-diagram.md` - Shows how different system components communicate with each other
- `activity-diagram.md` - Shows the step-by-step workflow for customers and admins

### Task_3 - OOP Examples
Contains Java code examples demonstrating object-oriented programming concepts.

**Files:**
- `productHierarchy.java` - Shows inheritance, abstraction, and polymorphism using Product classes (physical and digital products)

This repository provides a complete overview of the ShopHub e-commerce platform through:
- **Design Documentation**: UML diagrams explaining system structure and behavior
- **Code Examples**: Working Java code demonstrating OOP principles
